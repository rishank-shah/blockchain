# Get All Blocks in BLockchain

    GET /api/blocks
    
Returns a list of all blocks.

## Example
### Request

    GET http://localhost:3000/api/blocks

### Response
``` json
[
  {
    "timestamp": 1,
    "lasthash": "0",
    "hash": "0",
    "data": [
      
    ],
    "difficulty": 2,
    "nonce": 0
  }
]
```
